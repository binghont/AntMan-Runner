ABOUT

We used Cocos 2d – x as basic engine to build this game, used Xcode and mainly programmed with C++. We constructed the foundation and structure of thew game by creating and connecting different scenes, and add different layers respectively. Besides, we added our own physical engine for detect collision and running. For level design, we used Tiled Map to design maps and simulated scenes. This will make scenes more realistic and challenging through simulating different terrains and obstacles. Moreover, this bring our game with extendable property too.

We implemented speed up, fly, shrink, revive, energy, enemy features under this framework. In order to distinguish the power difference between small and big Ant - man, we designed two different jump distance for him. Also we add features like getting invincibility when speeding up, alerting before the enemy fire up etc, to balance the game difficulty and make this game more playable and durable.

We design themes and difficulty for different from easy to hard, which will make player learn how to control Ant – man and master skills in this game. For better looking of the scene in game, we also processed and rendered every blocks in the map and fit every block into grid.

Players can Interact and experience more when running as Ant - man. Each level has several different challenges for users to complete. We provided a multipath selection mechanism based on different risks, this will give the choice and space of making strategy to players. BTW, if you are good to find ester eggs, there are also some bonus in some level. Go and find them!
